# Artifacts

Generated files for AI review, debugging, and certification reports.

## Commit policy

| File | Commit? | Purpose |
|------|---------|---------|
| `dev_annotation_latest.json` | **Yes** when reporting bugs | Spray paint coordinates for agents |
| `dev_annotation_latest.png` | **Yes** with JSON | Screenshot context |
| `flow_state_proof_latest.txt` | No (local scratch) | Regenerated by proof tier |
| `kit_map_gap_audit_latest.json` | Optional | Gap audit output |

## Workflow

1. Spray bug in-game → Export
2. Commit JSON + PNG **or** attach to GitHub issue
3. Tell agent the file path in `CURRENT_FOCUS.md` or issue body
4. Clear spray after fix is verified

See [docs/BUG_REPORT_WORKFLOW.md](../docs/BUG_REPORT_WORKFLOW.md).

## Gitignore

Scratch artifacts are ignored; `dev_annotation_latest.*` are **tracked** when you commit them for agent review.
